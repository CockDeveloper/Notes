# Coding Convention

## Naming Convention
Letter case-separated words:
* camelCase
* PascalCase
* snake_case

## Constants 
1.Syntax: \<DOMAIN\>_\<NAME\>

2.Require:  UPPER_CASE

3.Example:
```objective-c
// Objecive-C
// https://developer.apple.com/documentation/uikit/uikit_constants
UIKIT_STRING_ENUMS
```
```swift
// Swift
// https://developer.apple.com/documentation/uikit/uikit_constants
var UIKIT_STRING_ENUMS: Int32
```
